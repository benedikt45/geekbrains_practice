Важная информация

Данный прибор предназначен только для домашнего использования. Прибор можно использовать только для переработки такого количества продуктов и в течение такого времени, которые характерны для домашнего хозяйства. Прибор предназначен для перемешивания, взбивания и смешивания мягких продуктов питания и жидкостей, а также для вымешивания мягкого теста. Прибор запрещается использовать для переработки других веществ или предметов.
Используйте прибор только внутри помещений при комнатной температуре на высоте не выше 2000 м над уровнем моря.
